Máquina de
Prospecção com IA

Do zero à mensagem personalizada em 15 minutos. Claude Code busca os leads no Google Maps, pesquisa cada empresa e escreve a abordagem certa para cada uma.

🤖 Claude Code 🗺️ Google Maps 📊 Instagram 📄 DOCX pronto 🐍 Python
1
Pré-requisito

O que você precisa antes

Você precisa de uma única coisa instalada. O resto é instalado automaticamente pelo script de instalação.

🤖 Claude Code obrigatório
O Claude Code é o único pré-requisito. Tudo mais — Python, Scrapling, Playwright, python-docx — é instalado automaticamente na próxima etapa.
→ claude.ai/code
✅ Isso é tudo. Se você já tem o Claude Code instalado, pode ir direto para a seção 02.
2
Uma vez só

Instalação automática

Você faz isso uma única vez. Depois, a máquina fica pronta para usar sempre que precisar.

1
Clonar o repositório — diga ao Claude Code
Abra o Claude Code e diga exatamente isso:
Comando para o Claude Code 
Clona o repositório github.com/amandadinizmkt/maquina-de-prospeccao e roda o instalar.py
Ou, se preferir rodar no terminal manualmente:
Terminal (alternativa manual) 
git clone https://github.com/amandadinizmkt/maquina-de-prospeccao.git
cd maquina-de-prospeccao
python3 instalar.py
2
O que o instalar.py faz automaticamente
O script verifica e instala tudo que a máquina precisa para funcionar:
  • ✓Python 3.10+ — verifica a versão instalada
  • ✓Scrapling — scraper que lê o Google Maps
  • ✓Playwright — abre e controla o navegador automaticamente
  • ✓python-docx — gera o arquivo DOCX com as mensagens prontas
3
Saída esperada no terminal
Quando tudo estiver instalado, você vai ver:
Saída do instalar.py 
✅ Python 3.14 — ok
✅ Scrapling 0.4.8 — já instalado
✅ Playwright Python — já instalado
✅ python-docx — já instalado
✅ Tudo instalado! Pronto para prospectar.
Não tem Python? Se aparecer erro de versão do Python, baixe o Python 3.10 ou superior em python.org/downloads. Depois repita o comando do Passo 1.
3
Toda vez que for prospectar

Buscar leads

Diga ao Claude Code o nicho e a cidade. Ele abre o Google Maps, rola a lista de resultados e salva tudo em uma planilha automaticamente.

Comando para o Claude Code 
Busca [tipo de empresa] em [cidade]

Exemplos reais:

Exemplos de busca 
Busca clínicas de estética em São Paulo
Busca escritórios de advocacia em Belo Horizonte
Busca lojas de beachwear em Florianópolis

O que acontece depois que você envia o comando:

1
Claude roda o prospectar.py
O script abre o Google Maps automaticamente no navegador, já com a busca pelo nicho e cidade que você escolheu.
2
Rola a lista e captura os resultados
O Playwright rola a lista de resultados automaticamente para carregar cerca de 30 empresas. Você pode ver o navegador abrindo — é normal.
3
Salva o CSV e abre a planilha
Um arquivo CSV é salvo com todas as informações de cada empresa. A planilha abre automaticamente no Excel ou Numbers para você ver os leads.
  • →Nome da empresa
  • →Avaliação no Google Maps (estrelas)
  • →Número de reviews
  • →Telefone
  • →Endereço completo
Dica de nicho: Quanto mais específico o nicho, melhor a qualidade dos leads. "Clínicas de estética" é melhor do que "empresas de saúde". "Advogados especializados em família" é melhor do que "escritórios de advocacia".
4
Passo 2 da prospecção

Escolher e pesquisar

Com a planilha aberta, você escolhe quais leads o Claude vai pesquisar a fundo. Você pode indicar os números ou deixar o Claude escolher os melhores.

Opção 1 — você escolhe 
Quero pesquisar os leads 3, 7 e 12
Opção 2 — Claude escolhe os melhores 
Você escolhe os 5 com mais potencial para contratar gestão de redes

O que Claude pesquisa em cada lead:

📸 Instagram
Seguidores, número de posts, quantas contas está seguindo, bio, frequência de publicação e qualidade aparente do conteúdo.
🌐 Site
Plataforma usada, se existe ou não, qualidade visual, velocidade aparente e presença digital geral.
🗺️ Google Maps
Avaliação geral, número de reviews, se o perfil está completo e se a empresa responde às avaliações dos clientes.
🔍 Gargalos
Claude identifica as fraquezas reais de cada empresa — os problemas que você pode resolver — e usa isso nas mensagens.

Exemplos de gargalos que Claude identifica:

"2.006 posts com apenas 11K seguidores — estratégia de conteúdo não converte"
"Seguindo 2.975 contas — tática de follow/unfollow prejudica o alcance orgânico"
"Nota 2,5 no Maps com 188 avaliações — crise de reputação sem gestão"
"Conta principal do Instagram fora do ar — presença digital comprometida"
"Sem site próprio — depende 100% de uma plataforma de terceiros"
💡
Por que isso importa: A mensagem de prospecção que menciona um problema real da empresa converte muito mais do que uma abordagem genérica. O Claude usa esses gargalos para criar uma abertura que mostra que você entende o negócio deles — antes de falar de preço.
5
Passo final

Gerar as mensagens

Após a pesquisa, o Claude escreve mensagens personalizadas para cada lead — com base nos gargalos que encontrou. Você recebe tudo num arquivo DOCX pronto para usar.

Comando para gerar as mensagens 
Gera as mensagens personalizadas para os leads pesquisados.
Meu nome é [seu nome], trabalho com [seus serviços] para [seu nicho].

O que você recebe no arquivo DOCX:

1
Uma seção por lead
O arquivo está organizado com o nome de cada empresa no topo, seguido de todas as informações encontradas na pesquisa.
2
Gargalos identificados
Um resumo dos problemas encontrados — é o que o Claude usou para personalizar a abordagem de cada empresa.
3
Mensagem de primeiro contato (WhatsApp / DM)
Máximo 3 linhas. Menciona algo específico da empresa, mostra que você entende o negócio e termina com uma pergunta aberta — sem forçar venda.
4
Follow-up (para 2-3 dias depois)
Uma mensagem diferente da primeira para enviar se não houver resposta — ainda sem forçar a venda.
5
E-mail com assunto
Assunto em até 7 palavras (sem clickbait) + corpo com apresentação, gancho específico e CTA suave. Máximo 5 linhas.
Para abrir o DOCX no Finder 
Abre o arquivo de mensagens no Finder
✦
Copie e use

Prompts prontos

Cole esses prompts no Claude Code para qualificação avançada, abordagem por canal e refinamento das mensagens geradas.

Qualificação — antes de escolher quem pesquisar

📊 Análise de potencial dos leads
"Analisa esses leads e me diz quais são os 5 com mais potencial para contratar gestão de redes sociais. Considera: avaliação no Maps, número de reviews, presença digital aparente e potencial de crescimento no nicho. Para cada lead, me dá: Score de 1 a 10 · Motivo em 1 frase · Canal sugerido (WhatsApp / Instagram DM / E-mail) · Prioridade (Alta / Média / Baixa)"

Abordagem por Canal

💬 WhatsApp / Instagram DM
"Para cada lead do TOP 5, cria: 1. Mensagem de primeiro contato (máx 3 linhas) — menciona algo específico da empresa (gargalo encontrado), mostra que você entende o negócio deles, termina com pergunta aberta, não com oferta. 2. Follow-up (para 2-3 dias depois sem resposta) — diferente da primeira, ainda sem forçar a venda. Meu nome é [nome]. Trabalho com [serviço] para [nicho]."
📧 E-mail
"Para cada lead, cria: Assunto (máx 7 palavras, sem clickbait) + Corpo (apresentação 1 linha + gancho específico + CTA suave). Máximo 5 linhas no corpo."

Refinamento de Mensagem

✏️ Reescrever com ajuste de tom
"Essa mensagem para [nome do lead] ficou muito [genérica / formal / longa]. Reescreve sendo mais [específica / casual / direta]. Mantém a menção ao gargalo de [descrever gargalo]."
