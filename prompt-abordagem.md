# Prompt de Script de Abordagem Personalizada

Use esse prompt depois de ter a lista qualificada para gerar mensagens de primeiro contato.

---

## PROMPT — WhatsApp / Instagram DM

```
Você é um especialista em copywriting para abordagem de vendas B2B.

Preciso de scripts de primeiro contato para as empresas abaixo. O contexto é:

- Quem sou: [SEU NOME], [SEU CARGO] na [SUA EMPRESA]
- O que ofereço: gestão de redes sociais (Instagram e/ou outras) + [OUTROS SERVIÇOS SE TIVER]
- Tom: direto, humano, sem ser robótico ou vendedor demais
- Canal: WhatsApp ou Instagram DM

Empresas para gerar script:

[COLE A TABELA COM AS EMPRESAS DO TOP 5 AQUI]

Para cada empresa, gere:
1. Uma mensagem de primeiro contato (máximo 3 linhas)
   - Mencione algo específico da empresa (nome, nicho, localização ou avaliação)
   - Mostre que você entende o negócio deles
   - Termine com uma pergunta aberta, não com uma oferta

2. Uma mensagem de follow-up (para enviar 2-3 dias depois se não responder)
   - Diferente da primeira
   - Ainda sem forçar a venda

Formate como:
**[Nome da empresa]**
📱 Primeiro contato: [mensagem]
🔁 Follow-up: [mensagem]
```

---

## PROMPT — E-mail

```
Você é um especialista em cold email para agências de marketing.

Preciso de e-mails de prospecção para as empresas abaixo. Contexto:

- Quem sou: [SEU NOME], [SEU CARGO] na [SUA EMPRESA]  
- O que ofereço: [SEUS SERVIÇOS]
- Tom: profissional mas humano, curto (máximo 5 linhas no corpo)

Empresas:

[COLE AS EMPRESAS AQUI]

Para cada empresa, gere:
- Assunto do e-mail (máximo 7 palavras, sem clickbait)
- Corpo do e-mail (apresentação de 1 linha + gancho específico + CTA suave)

Formato:
**[Nome da empresa]**
Assunto: [assunto]
Corpo:
[texto do e-mail]
```
